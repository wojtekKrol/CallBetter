## Getting started :octocat:

###  Prequalities
```
Node >= 12
yarn >= 1.22
docker >= 19
```

###  Clone repo and install dependencies
```
git clone git@github.com:wojtekKrol/CallBetter.git
cd CallBetter
yarn install
````

### Run server and client
<b>Server:</b> 
```
yarn start:server
```
<b>Client:</b> 
```
yarn start:client
```

### Or run server in docker-compose
```
docker-compose up --build
```
